# config/

Infrastructure configuration and agent capabilities. Human-authored, checked out with the repo, not personal.

## Contents

- **`mcp_servers.json`** — MCP server definitions passed to the `claude` CLI via `--mcp-config`. Generated by `scripts/setup.py` on first run (the `uv` path varies per machine).
- **`skills/`** — Skill fragment files (`*.md`). Each file defines a capability the agent can invoke by name. `scripts/setup.py` seeds this directory with bundled defaults; add your own alongside them.

## Ownership

Edit freely. Changes take effect on the next agent start (MCP config) or immediately (skills, which are read on demand).
