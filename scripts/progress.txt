## Codebase Patterns
- Use `.venv/bin/pytest` / `.venv/bin/ruff` / `.venv/bin/pyright` directly (uv run fails to find tools)
- Atomic file writes: write to `.json.tmp` then `os.replace()` to final path
- Tag interception pattern: regex sub returning "" + side effects (see memory:write, skill:imap, skill:schedule)
- croniter: pass `last_run` or epoch datetime as start, call `.get_next(datetime)` to find next fire time
