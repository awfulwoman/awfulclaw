# Daily Briefing Skill

Deliver a concise morning briefing to the user covering:

1. **Schedule** — summarise any scheduled reminders or tasks due today or soon
2. **Open Tasks** — highlight any incomplete tasks from memory
3. **Key Facts** — mention anything time-sensitive from stored facts
4. **Anything else** — surface any proactive notes worth raising

Keep the tone warm but brief. End with an encouraging word for the day ahead.
If there is nothing to report, say so in one sentence and wish them a good day.
